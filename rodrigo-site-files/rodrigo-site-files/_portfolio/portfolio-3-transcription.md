## Overview

Leading the development of real-time transcription and visualization pipelines to analyze classroom conversations and support research on effective classroom interactions at UMBC's Division of Information Technology.

## System Architecture

```
Audio Input → Transcription → Speaker ID → LLM Analysis → Visualization
                                              ↓
                                        AWS Redshift
                                              ↓
                                   Tableau / Power BI
```

## Technical Components

### 1. Audio Processing Pipeline
- Converts classroom recordings to structured text
- Real-time processing capabilities

### 2. Speaker Identification
- Distinguishes between instructors and students
- Tracks conversation flow and participation

### 3. LLM Integration
- Extracts insights and patterns from conversations
- Identifies effective teaching moments

### 4. Analytics Dashboard
- Presents findings to educational researchers
- Interactive visualizations in Tableau and Power BI

## Key Features

- Real-time processing capabilities
- Scalable cloud infrastructure on AWS
- Cross-functional Agile development
- Comprehensive visualization and reporting
