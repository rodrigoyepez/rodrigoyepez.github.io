We tackle 8-class multi-layer cloud classification using GOES ABI satellite data with CloudSat/CALIPSO ground truth. Using Random Forest with data augmentation methods (SMOTE, ADASYN, ROS), we achieved 83.6% classification accuracy for climate and weather prediction applications.
